
## build

build library in zlogsrc:
$ make

or 
modify $(CC) in makefile to arm-arago-linux-gnueabi-gcc before make


## usage

copy the following files to you project:

zlog.h
bzlog.a
libzlog.so
libzlog.so.1
libzlog.so.1.2

then use zlog API in you code.
